# Universal OCR with EasyOCR & NLP

Extract text from any printed image and enhance it using basic Natural Language Processing (NLP) for real-world applications like business card reading, document processing, or automating data entry.

## Features

- Universal OCR: Works on any printed image (business cards, posters, receipts, flyers, etc).
- NLP Augmentation: Cleans and corrects extracted text; highlights structured info (names, addresses, etc.).
- Language Support: Easily adapt to other languages by changing model parameters.
- Modular & Extensible: Simple, well-commented codebase is easy to extend for your own use-cases.
